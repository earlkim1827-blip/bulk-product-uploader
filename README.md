## 1. 개발 환경

### 언어 및 런타임
| 항목 | 버전 |
|------|------|
| Python | 3.13.3 (64-bit) |
| tkinter | 8.6 (Python 기본 내장) |

### 주요 패키지
| 패키지 | 버전 | 용도 |
|--------|------|------|
| pandas | 3.0.2 | 엑셀 데이터 파싱 |
| openpyxl | 3.1.5 | 엑셀 파일 읽기/쓰기 |
| requests | 2.33.1 | HTTP API 통신 |

### 개발 도구
- **OS**: Windows 11
- **IDE**: 자유 (VS Code 권장)
- **버전 관리**: Git + GitHub

### 설치 방법
```bash
# 패키지 설치
pip install -r requirements.txt

# 실행
python main.py
```

## 2. 아키텍처

### 전체 흐름

```
[엑셀 파일 (.xlsx)]
        │
        ▼
┌─────────────────────┐
│   excel_reader.py   │  엑셀 파싱 → Product 객체 리스트
└─────────────────────┘
        │
        ▼
┌─────────────────────┐
│  product_model.py   │  공통 상품 데이터 모델 + 유효성 검사
└─────────────────────┘
        │
        ▼
┌─────────────────────┐
│    uploader.py      │  멀티 플랫폼 업로드 관리 (비동기 스레드)
└─────────────────────┘
        │
   ┌────┴────┬──────────┬──────────┬──────────┐
   ▼         ▼          ▼          ▼          ▼
[스마트스토어] [쿠팡] [11번가] [지마켓] [옥션]
```

### 폴더 구조

```
대량등록솔루션/
│
├── main.py                   # 진입점 (로깅 설정 + GUI 실행)
├── requirements.txt          # 패키지 목록
├── config.json               # API 키 저장 (git 제외)
│
├── core/                     # 핵심 비즈니스 로직
│   ├── product_model.py      # 상품 데이터 클래스 (Product, ProductOption)
│   ├── excel_reader.py       # 엑셀 → Product 변환
│   └── uploader.py           # 멀티 플랫폼 업로드 관리
│
├── api/                      # 쇼핑몰 API 클라이언트
│   ├── base.py               # 공통 베이스 클래스 (BaseAPIClient)
│   ├── smartstore.py         # 네이버 스마트스토어 (OAuth2)
│   ├── coupang.py            # 쿠팡 (HMAC-SHA256)
│   ├── st11.py               # 11번가 (API Key)
│   ├── gmarket.py            # 지마켓 (ESM Plus)
│   └── auction.py            # 옥션 (ESM Plus)
│
├── gui/                      # 화면 구성 (tkinter)
│   ├── main_window.py        # 메인 화면
│   └── settings_window.py    # API 키 설정 화면
│
├── templates/
│   └── 상품등록_템플릿.xlsx  # 엑셀 입력 양식
│
└── logs/                     # 실행 로그 (자동 생성)
```

### 각 모듈 역할

| 모듈 | 역할 | 주요 클래스/함수 |
|------|------|----------------|
| `core/product_model.py` | 상품 데이터 구조 정의 | `Product`, `ProductOption` |
| `core/excel_reader.py` | 엑셀 파싱 | `read_excel()` |
| `core/uploader.py` | 업로드 오케스트레이션 | `Uploader` |
| `api/base.py` | API 공통 인터페이스 | `BaseAPIClient` |
| `api/*.py` | 각 플랫폼별 API 구현 | `register_product()` |
| `gui/main_window.py` | 메인 UI | `MainWindow` |
| `gui/settings_window.py` | API 키 설정 UI | `SettingsWindow` |

### 인증 방식 (플랫폼별)

| 플랫폼 | 인증 방식 | 필요 정보 |
|--------|----------|----------|
| 스마트스토어 | OAuth 2.0 (HMAC 서명) | Client ID, Client Secret |
| 쿠팡 | HMAC-SHA256 | Access Key, Secret Key, Vendor ID |
| 11번가 | API Key (헤더) | API Key |
| 지마켓 | App Key + Cert Key | App Key, Cert Key, 판매자 ID |
| 옥션 | App Key + Cert Key | App Key, Cert Key, 판매자 ID |

## 3. 배포 환경

### 지원 환경
| 항목 | 내용 |
|------|------|
| OS | Windows 10 / 11 (64-bit) |
| Python | 3.8 이상 |
| 네트워크 | 인터넷 연결 필수 (API 통신) |

### 실행 방법 (일반 사용자)

**방법 A — Python 직접 실행**
```bash
pip install -r requirements.txt
python main.py
```

**방법 B — 단일 실행파일 (.exe) 빌드** *(예정)*
```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name 대량등록솔루션 main.py
# dist/대량등록솔루션.exe 생성됨
```

### 설정 파일 위치
| 파일 | 위치 | 내용 |
|------|------|------|
| `config.json` | 프로그램 루트 | API 키 (자동 생성) |
| `logs/app.log` | 프로그램 루트 | 실행 로그 (자동 생성) |

> ⚠️ `config.json`에는 API 키가 저장됩니다. 절대 외부에 공유하지 마세요. (`.gitignore`에 포함됨)

## 4. 디자인 가이드

### 색상 팔레트

| 역할 | 색상 | HEX |
|------|------|-----|
| 헤더/강조 | 다크 네이비 | `#2C3E50` |
| 배경 | 라이트 그레이 | `#F5F5F5` |
| 흰 배경 | 화이트 | `#FFFFFF` |
| 텍스트 기본 | 다크 그레이 | `#555555` |
| 성공 | 그린 | `#27AE60` |
| 오류 | 레드 | `#E74C3C` |
| 경고 | 오렌지 | `#F39C12` |
| 로그 배경 | 다크 (코드 에디터) | `#1E1E1E` |
| 로그 텍스트 | 라이트 | `#D4D4D4` |

### 폰트

| 용도 | 폰트 | 크기 | 스타일 |
|------|------|------|--------|
| 타이틀 | 맑은 고딕 | 16pt | Bold |
| 서브타이틀 | 맑은 고딕 | 9pt | Normal |
| 본문/레이블 | 맑은 고딕 | 9~10pt | Normal |
| 로그 출력 | Consolas | 9pt | Normal (고정폭) |

### 레이아웃 구조

```
┌─────────────────────────────────────────┐
│         헤더 (타이틀 + 플랫폼 목록)        │  배경: #2C3E50
├─────────────────────────────────────────┤
│  1. 엑셀 파일 선택  [파일선택] [템플릿저장]  │
├─────────────────────────────────────────┤
│  2. 쇼핑몰 선택  ☑스마트스토어 ☑쿠팡 ...  │
├─────────────────────────────────────────┤
│  3. 상품 미리보기 (테이블)                 │
├─────────────────────────────────────────┤
│  4. 등록 진행 상황                         │
│  ████████████░░░░  진행바                │
│  [로그 출력 영역 - 다크 테마]              │
├─────────────────────────────────────────┤
│  [로그저장] [로그지우기]    [중단] [등록시작]│
└─────────────────────────────────────────┘
```

### UI 컴포넌트 규칙

- **LabelFrame**: 각 섹션을 구분하는 그룹 박스 사용
- **버튼**: 오른쪽 정렬, 주요 액션은 오른쪽 끝
- **테이블(Treeview)**: 세로 스크롤바 항상 표시, 오류 행은 빨간색
- **진행바**: 등록 시작 시 0→100% 자동 업데이트
- **로그창**: 다크 테마, 고정폭 폰트, 타임스탬프 `[HH:MM:SS]` 형식
